JAVADOC API1.8

About

The Javadoc Reloaded project transforms arbitrary Javadoc API folders into fully fledged searchable webapps from the 21st century. It's a proposal for JEP 225 crafted by Benjamin Winterberg.

The source code is hosted on GitHub and published under the MIT license. Feel free to fork this project and send me pull requests. You can also send me feedback by opening an issue.

Follow me on Twitter or GitHub for more java-related stuff. You might also be interested in reading my blog.

Search queries

Search Description
action Finds every class which name contains action. This is a shortcut for name:action
version:1.8 Finds every class which exists since version 1.8 (alias: since)
extends:runnable Finds every class which extends (or implements) runnable (alias: sub)
in:java.lang Finds everything from package java.lang (alias: package, pkg, from)
is:interface Finds every class which is an interface. Accepted values: class, interface, enum, annotation, abstract, final, functional
stream in:util is:interface since:8 A combined example query which should be self explanatory
