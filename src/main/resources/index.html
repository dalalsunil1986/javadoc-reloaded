Search Query Description
action Finds every class which name contains action. This is a shortcut for name:action
version:1.8 Finds every class which exists since version 1.8 (alias: since)
extends:runnable Finds every class which extends (or implements) runnable (alias: sub)
in:java.lang Finds everything from package java.lang (alias: package, pkg, from)
is:interface Finds every class which is an interface. Accepted values: class, interface, enum, annotation, abstract, final, functional
